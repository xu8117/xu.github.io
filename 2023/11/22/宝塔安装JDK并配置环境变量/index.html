宝塔安装JDK并配置环境变量

xu8117
2023-11-22

首先在/etc/profile.d目录下新建一个java.sh脚本（或者使用宝塔更方便）

sudo touch /etc/profile.d/java.sh

然后给该文件赋上可写和可执行权限

sudo chmod a+wx /etc/profile.d/java.sh
执行以下命令进入vim编辑脚本

sudo vi /etc/profile.d/java.sh
按i，进入编辑模式，输入以下两行代码

export JAVA_HOME=/usr/java/jdk1.8.0_121 //jdk安装路径
export PATH=$PATH:$JAVA_HOME/bin
按esc退出编辑，输入:wq!保存并退出
运行脚本，使用如下命令进入路径：

cd /etc/profile.d/
然后使用source命令执行该脚本

source java.sh
在任意目录下，输入java版本，发现java环境运行正常。
